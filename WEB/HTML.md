#### HTML

* HTML (Hyper Text Markup Language)
  
  * 웹 페이지를 구성하는 언어
  
  * Markup Language : 구조화된 언어 -> 각각의 요소들을 태그로 구조화 시킨 언어
  
  * -> A라는 태그를 보면 이런 의미를 가지고 있구나 해석할 수 있게끔 함
